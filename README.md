**PIXEL VAULT** 使用说明
为了防止数据损失，请下载下来-打开index.html进行使用，github的网页仅作展示。
**1. 核心存储**

- **本地化**：基于浏览器 `localStorage`，无服务器，数据仅保存在用户本地。
- **单文件**：所有代码集成在一个 `.html` 文件中，即开即用。

**2. 安全防护**

- **防君子锁 (PIN Lock)**：打开网页时强制全屏锁定，需输入 PIN 码解锁（支持重置）。
- **隐私掩码**：密码默认显示为 `******`，点击 `SHOW` 切换查看/隐藏。

**3. 数据管理**

- **增删查**：快速添加账户、一键删除条目。
- **JSON 备份/恢复**：支持一键导出数据为 JSON 文件，或从 JSON 文件恢复数据（方便跨设备迁移）。

**4. 效率工具**

- **一键复制**：点击 `COPY` 按钮直接复制密码到剪贴板。
- **实时搜索**：顶部搜索栏支持按服务名或用户名过滤，即输即显。
- **强密码生成**：内置随机生成器，生成 **14位** 包含 **大小写+数字+@** 的高强度密码。
